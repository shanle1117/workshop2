# Images Directory

## Logo File

Please place the UTeM logo image file here with the filename: `utem-logo.png`

The logo should be:
- PNG format (recommended for transparency)
- Circular logo with UTeM branding
- Recommended size: 200x200px or larger (will be scaled to 50px height in the header)

## File Location

The logo file should be named: `utem-logo.png`

Once the file is added, the logo will automatically appear in:
- Main website header (`index.html` and `main.html`)
- Admin dashboard header (`admin.html`)
